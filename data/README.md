# Data

Large single-cell datasets are not included in this repository.

The analysis uses processed perturbation single-cell RNA-seq datasets in
AnnData `.h5ad` format. Users should place the required files in this
directory before running the notebooks.

Expected files may include:

- AdamsonWeissman2016_GSM2406675_10X001.h5ad
- AdamsonWeissman2016_GSM2406681_10X010.h5ad

The data directory is excluded from Git because the files are large.
